## How to Add Badges

1. **After first GitHub push**, go to your Actions tab
2. **Click on a workflow** (e.g., CI)
3. **Click the three dots** (⋯) → "Create status badge"
4. **Copy the markdown** and add to README

Or use [shields.io](https://shields.io/) to create custom badges.
